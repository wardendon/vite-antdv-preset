# READ ME

- 安装依赖
```
npm install
```
- 调试
```
npm run dev:local
// npm run dev:test
// npm run dev:mock
```
- build
```
npm run build:prod
npm run preview
```
---

- 状态保持：[Pinia](https://pinia.vuejs.org/)
- UI库：[Ant Design Vue](https://www.antdv.com/docs/vue/introduce-cn/)
---
